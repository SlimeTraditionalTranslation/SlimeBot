# 安裝方式
只需將 jar 放入你的 ``plugins`` 資料夾中，並將伺服器開啟產生插件設定資料夾，在 ``plugins/Slimebot/config.yml`` 中填寫機器人的 Token 與伺服器 ID 即可完成。
